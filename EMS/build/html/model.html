class ems_prog.model.EmsModel(db_path)[source]

Bases: object

Manages the interaction with the SQLite database for employee information.

This class provides an interface for database operations and implements the Observer pattern to notify registered observers about changes in the data model.

db_path

Path to the SQLite database file.

Type:

str

con

Active database connection object.

Type:

sqlite3.Connection

cursor

Cursor object for executing SQL commands.

Type:

sqlite3.Cursor

observers

List of observer objects to be notified of changes.

Type:

list

notify_observers()[source]

Notifies all registered observers of a change in the data model.

Each observer’s update method is called to handle the notification.

update_data()[source]

Simulates a change: Modifies an existing record or adds a new one.

Notifies the Observer.

Logs the changes made.

create_table()[source]

Creates the ‘Empleados’ table in the database if it does not exist.
